# Nexbit Junior – Next.js App (App Router)

- Dev: npm run dev (port 3001)
- Build: npm run build
- Start: npm run start (port 3001)

Open http://localhost:3001 to view. Edit content in app/page.tsx and components/*; changes hot-reload.

Tech: Next.js 14 (App Router, TypeScript) + Tailwind CSS. Fonts and metadata are set in app/layout.tsx.
